# DevCamper Api

> Backend Api for DevCamper applivation,
  which is bootcamp directory website

## Usage

  Rename "config/config.env.env" to
  "config/config.env" and update the
  values/settings to your own

## Install Dependencies 

  mpm install

## Run App

# Run in dev mode
  mpm run dev

# Run prod mode
  npm start
